# Monte Carlo Integrator

This repository provides a Python implementation of a Monte Carlo Integrator, designed to estimate the area under a function's curve. The `FunctionIntegrator` class allows for Monte Carlo integration, visualizes the function and its integral, and supports parameter grids for multiple iterations. It enables efficient integration, including error analysis when an analytical solution is available.

## Features
- **Monte Carlo Integration**: Estimates the integral (area under the curve) of any given function.
- **Visualization**: Plots the function and highlights the area under the curve.
- **Parameter Grid**: Perform integration over a range of iterations and analyze the results.
- **Error Analysis**: Compare Monte Carlo estimates with an analytical solution (if provided).
